AI-Powered Smart Asset Management System

Enterprise-grade asset management platform scaffold.

Quick start:

- Copy `.env.example` to `.env` and fill values.
- Start services with `docker compose up --build` (requires Docker). This now runs the Node.js backend in `backend-node/`.
- Frontend: `cd frontend && npm install && npm run dev`
- Backend: `cd backend-node && npm install && npm run dev`

See docs/ for detailed design and schema.
